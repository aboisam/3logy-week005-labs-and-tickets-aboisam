# Week 5 - React Fundamentals

**Duration:** Monday-Friday, Week 5  
**Focus:** React Foundations — Components, State, Hooks, Forms  
**Sprint:** Building toward Sprint 03 (React Task Manager Application)

---

## 🎯 LEARNING OBJECTIVES

By the end of Week 5, students will be able to:
- ✅ Understand component-based architecture and React's declarative paradigm
- ✅ Create functional components with JSX syntax
- ✅ Pass data between components using props
- ✅ Manage component state with the useState hook
- ✅ Handle side effects with the useEffect hook
- ✅ Build forms with controlled components and validation
- ✅ Use modern JavaScript (ES6+) features: arrow functions, destructuring, spread operator

## 📅 WEEKLY SCHEDULE

### Monday (2:00-5:00 PM Physical)
**Topic:** Introduction to React, JSX, Components & Props  
- Lecture: What is React? Component-based architecture, JSX syntax
- Demo: Creating components, passing props, rendering lists
- **Lab 17:** Component Gallery — Build 4+ reusable React components
- **Assigned:** TICKET-17 (React Profile Page)

### Tuesday (2:00-5:00 PM Virtual)
**Topic:** State Management with useState  
- Lecture: State vs props, the useState hook, immutable updates
- Demo: Counter, toggles, todo list with state
- **Lab 18:** Interactive Todo App — Full CRUD with filtering
- **Assigned:** TICKET-18 (Counter Component Collection)

### Wednesday (2:00-5:00 PM Physical)
**Topic:** useEffect & Component Lifecycle  
- Lecture: Side effects, dependency arrays, cleanup functions
- Demo: Timers, data fetching, document title updates
- **Lab 19:** Timer & Data Fetcher — useEffect patterns
- **Assigned:** TICKET-19 (Weather Dashboard)

### Thursday (2:00-5:00 PM Physical)
**Topic:** Forms & Controlled Components  
- Lecture: Controlled vs uncontrolled inputs, form validation
- Demo: Registration form, multi-step form, validation patterns
- **Lab 20:** Registration Form — Complete form with validation
- **Assigned:** TICKET-20 (Survey Builder)

### Friday (2:00-5:00 PM Physical)
**Topic:** Sprint 03 Planning — React Task Manager  
- Sprint 03 requirements review
- Component tree planning & wireframing
- Project setup with create-react-app
- **All Week 5 labs/tickets due by 5:00 PM**
